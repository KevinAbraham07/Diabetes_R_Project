## 📌 Overview

This project investigates the relationship between **lifestyle factors** — BMI, physical activity, smoking, diet, and alcohol consumption — and **diabetes risk** using a balanced 50/50 split dataset from the **Behavioral Risk Factor Surveillance System (BRFSS) 2015** survey.

It includes two main components:

| Component | Description |
| :--- | :--- |
| 🖥️ **Interactive Shiny Dashboard** | Web-based dashboard with dynamic charts and summary stats |
| 📊 **Advanced Analysis Pipeline** | 28-step R script covering EDA, multi-model training, and validation |

## 📊 Dataset

> **Source:** [CDC BRFSS 2015 — Kaggle](https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset)
>
> **Records:** ~70,000 (balanced — 50% diabetic, 50% non-diabetic) &nbsp;|&nbsp; **Features:** 22 health indicators

<details>
<summary><strong>📋 Key Variables (click to expand)</strong></summary>
<br />

| Variable | Description |
| :--- | :--- |
| `Diabetes_binary` | 0 = No Diabetes, 1 = Diabetes / Prediabetes |
| `BMI` | Body Mass Index |
| `PhysActivity` | Physical activity in the past 30 days (0/1) |
| `Smoker` | Smoked ≥ 100 cigarettes in lifetime (0/1) |
| `Fruits` | Consumes fruit ≥ 1 time per day (0/1) |
| `Veggies` | Consumes vegetables ≥ 1 time per day (0/1) |
| `HvyAlcoholConsump` | Heavy alcohol consumption (0/1) |
| `Age` | Age category (1–13) |

</details>

## ✨ Features

### 🖥️ Shiny Dashboard — `app.R`

- **📦 Summary Value Boxes** — Total records, diabetes cases, and non-diabetic counts
- **📊 Diabetes Distribution** — Visual balance of the dataset
- **🔀 Lifestyle Factor Explorer** — Interactive charts for BMI, Smoking, Alcohol, and Diet
- **👤 Age vs Diabetes** — Demographic trends analysis
- **🔥 Correlation Heatmap** — Relationship analysis between all 22 indicators

### 📈 Analysis Pipeline — `scripts/diabetes_analysis.R`

A comprehensive **28-step pipeline** including:
- **🛠️ Data Wrangling**: Outlier detection (BMI), factor conversion, and missing value checks.
- **🧪 Statistical Testing**: Chi-Square tests and T-tests for significance.
- **🧠 Predictive Modeling**: Logistic Regression and Random Forest implementations.
- **📉 Advanced Validation**: 5-Fold Cross-Validation and Hyperparameter Tuning.
- **⚖️ Fairness Analysis**: Bias checks across demographic subgroups (e.g., Sex).
- **📦 Deployment Ready**: Model serialization (RDS) and a custom `predict_diabetes_risk()` function.

## 🎯 Usage

### Run the Shiny Dashboard

```r
shiny::runApp("app.R")
```

### Execute the Full Analysis Pipeline

```r
source("scripts/diabetes_analysis.R")
```

The script will execute all 28 steps, display visualizations (using `patchwork` and `pROC`), and save the final model to the `models/` directory for production use.
